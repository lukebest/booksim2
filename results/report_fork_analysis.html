Mesh 16x16，N=256，H=4 V=6 cycle/link。 下 ramp 带宽：1 或 2 flit/cycle/node（单向原生=1，双向原生=2）。 更新 2026-06-25T06:45:35.897147+00:00。 数据 sweep_buffer_pareto.py → buffer_pareto_16x16.json。

模型概要

  • §1 严格 0-buffer：router 内无排队；border/短弧方案允许边界 AFIFO ≤ 5 flit（6 cy 链路延迟）；调度器 sched_ring_zerobuf 或刚性 sched_zerobuf_compare.pack。
  • §2 允许 router 缓冲：冲突无关流水线 TDM 日历；每输出 port link_buf ≤ 6 flit；下 ramp 突发缓冲 ramp_buf ∈ [0,6] flit（持续 eject 仍 ≤ ramp_bw flit/cy，缓冲仅吸收到达突发）。
  • 方案族：ring / quad-center / border / multitree / hybrid B=k / grid Qx×Qy（与 Pareto sweep 一致）。
